### 1.3 standalone 정책 (DevHub umbrella §1.2 G7 흡수)

본 repo 는 **완전 standalone** 운영:

- 다른 backend (DevHub `backend-core` 등) 연결 ❌
- OIDC ❌ (외부 vault 의 PAT/API key 만 사용)
- Network: 3 단계 (dev 사외 / staging 사내 / production 사내)
- Storage: file-based (M-v0.3.0+ default) + DB-based 듀얼 모드 (M-v0.3.2+ 옵션)
- Tier: **사외** (사내 한정 정보 0)
- API versioning: `/api/v0-2/` prefix (M-v0.3.4+ 부터 `/api/v0-3/` dual endpoint)
